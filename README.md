# Stall Monitor

An IOT application to monitor the occupancy of the Men's restroom at work.
 
## Getting Started

### Clone and install
`git clone "https://github.com/dasavi/stallmonitor.git"`
`npm install`

### Start Node server
`node index.js`

### Start grunt watcher for scss files
`grunt watch`

### Goto:
`http://localhost:5000`
